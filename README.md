Traditional digital learning platforms often deliver generic, non-adaptive content and lack meaningful engagement features.
This project introduces an AI-assisted, teacher-supervised e-learning ecosystem that delivers personalized micro-learning experiences with curriculum alignment, adaptive pathways, and real-time motivational analytics.
